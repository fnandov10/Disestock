# DiseStock

Sistema de gestión de inventario desarrollado como proyecto de formación del SENA.

## Módulo implementado

Gestión de Usuarios.

## Tecnologías utilizadas

- Java
- JDBC
- MySQL
- Git
- GitHub

## Funcionalidades

- Registrar usuarios.
- Consultar usuarios.
- Actualizar usuarios.
- Eliminar usuarios.
